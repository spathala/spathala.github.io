Website Feature's Navigation Instructions


Welcome to the user manual for the OFD WELLNESS application.This page offers easy and intuitive navigation help to non-technical users of the tool.


The OFD WELLNESS system is an interactive web-based application that Omaha Fire Department's Saftey and Wellness Division staff can use to track and view the health records of the fire personnel. This application majorly tracks injuries, tuberculosis tests, immunizations, fit tests, accidents, biological and hazmat exposures.
